# useHoldingStream

`useHoldingStream()`

获取当前登录的用户的持有的资产数据, 会实时更新。

## import

```ts copy
import { useHoldingStream } from "@orderly/hooks";
```

## Usage

```ts copy
// get account info
const { data, loading, error } = useHoldingStream();
```

**Returns**

**`data`**

Type: `API.Holding[]`

**`usdc`**

Type: `number`

当前账户的 USDC 余额

**`loading`**

Type: `boolean`

**`error`**

Type: `Error`
