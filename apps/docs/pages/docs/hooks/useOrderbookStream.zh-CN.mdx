# useOrderbookStream

`useOrderbookStream(symbol:string, options?: { level: number })`

## import

```ts copy
import { usePrivateQuery } from "@orderly/hooks";
```

## Usage

```ts copy
// get account info
const [data, { onDepthChange, isLoading, onItemClick, depth, allDepths }] =
  useOrderbookStream(symbol, undefined, {
    level: 7,
  });
```

**Parameters**

- `symbol` - the symbol to get the orderbook for

**Returns**

- `onDepthChange()` - function to update the depth level
- `onItemClick()` - function to update the depth level
