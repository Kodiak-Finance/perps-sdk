# Hooks

`@orderly.network/hooks`

The hooks libraries provides a series of React hooks for React developers to ensure a seamless integration experience through the Orderly SDK. This library is a package built on top of [`@orderly.network/core`](/apis/modules/__orderly_network___core) and [`@orderly.network/perp`](/apis/modules/__orderly_network___prep).

## Coding standards

- If hooks returns an array, then the first elemnet will be the actual data, and the second element will be an object containing some supporting functions.
- All hooks ending with stream will update in real-time, via relevant websocket subscriptions.
- All hooks ending with query is a wrapper on API requests.

## List of hooks

### Base & Common

Provides basic hooks to help handle UI states, and some common state management logic.

- [useQuery](/apis/modules/__orderly_network___hooks/useQuery) - `GET` Public API
- [usePrivateQuery](/apis/modules/__orderly_network___hooks/usePrivateQuery) - `GET` Private API
- [usePrivateLazyInfiniteQuery](/apis/modules/__orderly_network___hooks/usePrivateLazyInfiniteQuery) - A version of `usePrivateQuery` that supports infinite scrolling
- [useLazyQuery](/apis/modules/__orderly_network___hooks/useLazyQuery)
- [useMutation](/apis/modules/__orderly_network___hooks/useMutation) - `POST` APIs
- [useWS](/apis/modules/__orderly_network___hooks/useWS) - WebSocket client
- [useConfig](/apis/modules/__orderly_network___hooks/useConfig) - An instance of the [ConfigStore](/apis/modules/__orderly_network___hooks/ConfigStore)
- [useEventEmitter](/apis/modules/__orderly_network___hooks/useEventEmitter) - An instance of the [EventEmitter](/apis/modules/__orderly_network___hooks/EventEmitter)
- [useLocalStorage](/apis/modules/__orderly_network___hooks/useLocalStorage) - Management (CRUD) of localStorage
- [useSessionStorage](/apis/modules/__orderly_network___hooks/useSessionStorage) - Management (CRUD) of sessionStorage
