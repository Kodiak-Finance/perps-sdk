# useOrderEntry

`useOrderEntry = ( symbol: string, side: OrderSide, reduceOnly: boolean = false, options?: UseOrderEntryOptions ): OrderEntryReturn`

创建订单

## Usage

## Parameters

`symbol`

Type: string

交易对

`side`

Type: OrderSide

下单方向,可能值: Buy/Sell

`reduceOnly`

Type: boolean

`options`

Type: any

## Returns

Type: Object

`maxQty`

Type: number

最大可下单数量

`freeCollateral`

Type: number

可用保证金
