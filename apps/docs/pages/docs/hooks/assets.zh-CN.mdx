# 资产

`@orderly.network/hooks` 提供了以下 hook，用于获取用户资产数据，可以视不同情况选择使用。

- [`useCollateral`](/apis/modules/__orderly_network___hooks/useCollateral) - 获取用户保证金
- [`useMaxQty`](/apis/modules/__orderly_network___hooks/useMaxQty) - 获取用户当前可下单的最大数量（基于 Symbol,下单方向，杠杆倍数等条件计算）
- [`useWithdraw`](/apis/modules/__orderly_network___hooks/useWithdraw) - 出金时获取用户可提取的资产,参考[出金](/apis/modules/__orderly_network___hooks/withdraw)
- [`useDeposit`](/apis/modules/__orderly_network___hooks/useDeposit) - 入金时获取用户钱包里 token 的余额，参考[入金](/)

用户资产计算的相关函数由[`@orderly.network/perp`](/)提供。
