# 行情

`@orderly.network/hooks`提供以下行情相关的 hook：

- [`useMarketsStream`](/apis/modules/__orderly_network___hooks/useMarketsStream) - 获取上架的交易对列表及其行情数据
- [`useMarkPrice`](/apis/modules/__orderly_network___hooks/useMarkPrice) - 获取单个交易对的标记价格
- [`useMarkPricesStream`](/apis/modules/__orderly_network___hooks/useMarkPricesStream) - 获取所有交易对的标记价格
- [`useMarketTradeStream`](/apis/modules/__orderly_network___hooks/useMarketTradeStream) - 获取单个交易对的历史成交记录
