# Hooks

`@orderly.network/hooks`

针对 React 开发者,我们提供了一系列的 React hooks, 以便于你在 React 项目中更方便地使用 Orderly SDK 的功能, `@orderly.network/hooks`是对[`@orderly.network/core`](/apis/modules/__orderly_network___core) [`@orderly.network/perp`](/apis/modules/__orderly_network___prep)的再次封装。

## 代码规范约定

- 如果 hook 返回的类型是数组，那么通常第一个元素是数据，第二个元素是一个 Object，包含一些辅助函数。
- 以 `stream` 结尾的 hook 表示是会实时更新的, 通常是包含了对相关 ws 的订阅。
- 以`query`结尾的 hook 是对网络请求的封装。

## Hooks 列表

### Base & Common

提供基础的, 与具体业务无关的 hooks, 帮助处理 UI 状态, 以及一些常用的状态管理逻辑。

- [useQuery](/apis/modules/__orderly_network___hooks/useQuery) - `GET`
- [usePrivateQuery](/apis/modules/__orderly_network___hooks/usePrivateQuery) - `GET` Private API
- [usePrivateLazyInfiniteQuery](/apis/modules/__orderly_network___hooks/usePrivateLazyInfiniteQuery) - 支持无限加载分页查询的 usePrivateQuery
- [useLazyQuery](/apis/modules/__orderly_network___hooks/useLazyQuery)
- [useMutation](/apis/modules/__orderly_network___hooks/useMutation) - `POST`
- [useWS](/apis/modules/__orderly_network___hooks/useWS) - WebSocket client
- [useConfig](/apis/modules/__orderly_network___hooks/useConfig) - [ConfigStore](/apis/modules/__orderly_network___hooks/ConfigStore)的实例
- [useEventEmitter](/apis/modules/__orderly_network___hooks/useEventEmitter) - [EventEmitter](/apis/modules/__orderly_network___hooks/EventEmitter)的实例
- [useLocalStorage](/apis/modules/__orderly_network___hooks/useLocalStorage) - localStorage 增删改查
- [useSessionStorage](/apis/modules/__orderly_network___hooks/useSessionStorage) - sessionStorage 增删改查
