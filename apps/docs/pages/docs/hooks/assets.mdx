# Assets

`@orderly.network/hooks` provides the following hooks to get data regarding a user's assets. This can be used under different use cases.

- [`useCollateral`](/apis/modules/__orderly_network___hooks/useCollateral) - retrieves information about a user's collateral
- [`useMaxQty`](/apis/modules/__orderly_network___hooks/useMaxQty) - retrieves the maximum quantity a user can place based on the symbol, side, leverage etc.
- [`useWithdraw`](/apis/modules/__orderly_network___hooks/useWithdraw) - retrieves the amount available for withdrawal, see [withdraw](/apis/modules/__orderly_network___hooks/withdraw)
- [`useDeposit`](/apis/modules/__orderly_network___hooks/useDeposit) - retrieves a user's token balance in their wallet when depositing, see [deposit](/apis/modules/__orderly_network___hooks/deposit)

Functions regarding asset calculations are provided by [`@orderly.network/perp`](/apis/modules/__orderly_network___perp)
