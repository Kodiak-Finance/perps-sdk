# useChains

`useChains(network, options)`

获取支持的链列表。

## Usage

```typescript
//import
import { useChains } from "@orderly.network/hooks";
```

```typescript
const [chains, { findByChainId }] = useChains("mainnet");

const arbInfo = findByChainId(42161);
```

## Parameters

### networkId

Type: string

可选值：`mainnet`、`testnet`

### options

Type: object

## Returns

Type: Array

### `[0]`

Type: Chains[]

Orderly 平台支持的链列表
