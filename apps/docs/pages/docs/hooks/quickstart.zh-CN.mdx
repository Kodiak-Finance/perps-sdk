# Quick Start

仅需二步，即可在你的项目中使用`@orderly.network/hooks`。

<Steps>

### Installation

Inside your React project directory, run the following:

```sh npm2yarn copy
npm install @orderly.network/types @orderly.network/hooks
```

### 添加 `OrderlyProvider`

在你的项目中, 添加 `OrderlyProvider` 组件, 通常包装根组件。

<Tabs items={['使用默认配置', '自定义']}>
  <Tabs.Tab>

在新建的项目中，建议使用默认配置，只需要提供你的 brokerId ([如何获取?](/)) 即可。

```tsx
import type { FC, PropsWithChildren } from "react";
import { OrderlyConfigProvider } from "@orderly.network/hooks";

const brokerId = "<your id>";

export const App: FC<PropsWithChildren> = ({ children }) => {
  return (
    <OrderlyConfigProvider brokerId={brokerId} networkId="mainnet">
      {children}
    </OrderlyConfigProvider>
  );
};
```

</Tabs.Tab>
  <Tabs.Tab>
  如果项目已经存在并且已经实现了Orderly Key的创建流程，或者其他需要自定义配置的情况，可以给 `OrderlyConfigProvider` 的 `configStore`/`keyStore` 属性传入自定义的配置, 参考[这里](/docs/hooks/faq)。

```tsx
import type { FC, PropsWithChildren } from "react";
import { OrderlyConfigProvider } from "@orderly.network/hooks";
import type {ConfigStore,OrderlyKeyStore} from "@orderly.network/core";

// 简化后的代码
const myConfigStore: ConfigStore = new <Your ConfigStore class>;
const myKeyStore: OrderlyKeyStore = new <Your OrderlyKeyStore class>;

export const App: FC<PropsWithChildren> = ({ children }) => {
const brokerId = "<your id>";
return (
  <OrderlyConfigProvider configStore={myConfigStore} keyStore={myKeyStore} networkId="mainnet">
    <children />
  </OrderlyProvider>
  );
};
```

</Tabs.Tab>
  
</Tabs>

### 完成!

现在你可以在你的项目中使用 `@orderly.network/hooks` 提供的所有 hooks 了。

```tsx
import { useAccount } from "@orderly.network/hooks";

export const UserInfo = () => {
  const { account, state } = useAccount();

  return (
    <div>
      <div>Address: {state.address}</div>
    </div>
  );
};
```

</Steps>

## mainnet / testnet

`OrderlyConfigProvider` 组件的 `networkId` 属性可以设置为 `mainnet` 或者 `testnet`，默认为 `mainnet`, `testnet`目前仅支持 Arbitrum Goerli。

```typescript {8}
import type { FC, PropsWithChildren } from "react";
import { OrderlyConfigProvider } from "@orderly.network/hooks";

const brokerId = "<your id>";

export const App: FC<PropsWithChildren> = ({ children }) => {
  return (
    <OrderlyConfigProvider brokerId={brokerId} networkId="testnet">
      {children}
    </OrderlyConfigProvider>
  );
};
```
