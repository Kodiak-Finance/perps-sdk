# Asset

Asset 类用于管理 Account 资产，提供余额查询、出入金等方法。

- deposit
- withdraw

```ts copy
const balance = await account.assetsManager.getNativeBalance({
  decimals,
});
```

<Callout>You do not need to manually initialize an `Asset` instance.</Callout>

{/* ## Instance properties */}

## Instance methods

### getNativeBalance

`getNativeBalance(options?: { decimals?: number }): Promise<string>`

获取 Native token 的余额

```ts copy
const balance = await account.assetsManager.getNativeBalance({
  decimals,
});
```

### getBalance

`getBalance( address?: string, options?: { decimals?: number; } ): Promise<string> `

获取指定 figible token 的余额

```ts copy
const balance = await account.assetsManager.getNativeBalance({
  decimals,
});
```

**Parameters**

- `address` string
- `options` object
  - `decimals` number

**Returns**

- `Promise<string>`
