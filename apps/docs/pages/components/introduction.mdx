

# Introduction

`@orderly.network/react` 提供了构建去中心化交易所所需要的一切组件，能够帮助你快速构建一个去中心化交易所。


## 特点

- 提供功能完整的React组件，可以零代码构建一个去中心化交易所；
- 交互友好，完美兼容移动端；
- 模块化设计，分层架构，可以根据自己的要求按需使用；
- 支持多语言；
- 主题可定制；


## 架构

![](/images/orderly_architecture.png)

`@orderly.network/react` 采用分层设计，分为基础 UI 组件、业务组件和页面三层，每一层都可以单独使用。
- UI components: 提供基础的 UI 组件，如按钮、输入框、表格等；
- Block components: 提供业务相关的组件，如交易对选择器、订单薄、交易历史等；
- Page: 提供完整的页面，如交易页面；







## 兼容性

- 现代浏览器和 IE11 及以上版本；
