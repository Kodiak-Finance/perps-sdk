import { LiveProvider, LiveEditor, LivePreview, LiveError } from "react-live";
import {Deposit,AssetsProvider,Page,Button,useModal,modal} from '@orderly.network/react'


# Deposit block component

入金组件，用于用户入金操作。

## 特点

- 组件内部已经集成了`@orderly.network/hoooks`, 无需任何参数即可使用；
- 组件内置钱包连接，用户状态检查等；
- 必须包含在`OrderlyAppProvider`,`ConnectorProvider`组件内使用;

## Usage

### Static mode

你可以在你的项目中直接展示`Deposit`组件：


<LiveProvider code={`<Deposit/>`} scope={{AssetsProvider,Deposit,Page}}>
<div className="flex justify-center py-3 mt-5 rounded shadow-lg bg-base-700">


<LivePreview />

</div>
  
  <LiveError />
</LiveProvider>

```typescript copy
import { Deposit } from "@orderly.network/react";

export const App = () => {
  return (
    <Deposit />
  );
};
```

### Modal模式

通过对话框的模式展示`Deposit`组件：

<LiveProvider code={`()=>{
    return (<Button onClick={()=>{
            dialog({
                title:'Deposit',
                content:<Deposit/>
            });
        }}>Open deposit dialog</Button>)
}`} scope={{AssetsProvider,Deposit,Page,Button,dialog:modal.dialog}}>
<div className="flex justify-center py-10 mt-5 rounded shadow-lg bg-base-700 color-item">

<LivePreview />

</div>
  
  <LiveError />
</LiveProvider>


```typescript copy
import { AssetsProvider, Deposit, modal } from "@orderly.network/react";

const { dialog } = modal;

export const App = () => {
  return (
    <Button
      onClick={() => {
        dialog({
          title: "Deposit",
          content: <Deposit />,
        });
      }}
    >
      Open deposit dialog
    </Button>
  );
};
```

{/* ## AssetsProvider

`AssetsProvider` 组件提供了跟资产相关的数据和操作，你可以在需要的地方用它包裹组件 */}

## APIs


### onOk

- Type: (result)=>void
- Required: false

入金完成后的回调函数，返回入金结果。
