# Orderly Connector

<Subtitle>@orderly.network/ui-connector</Subtitle>

```shell
npm install @orderly.network/ui-connector
```

The Orderly Connector is a React component that provides a simple way to connect to the Orderly Network.

Connector组件实现了以下功能：

- Create Orderly account
- Create an Orderly key


<Primary/>

## Import

```jsx
import {WalletConnectorWidget} from '@orderly.network/ui-connector';
// or
import {WalletConnectContent} from '@orderly.network/ui-connector';
```
