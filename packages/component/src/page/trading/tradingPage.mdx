# TradingPage

`TradingPage` 组件是一个完整的交易页面维护，基本涵盖了当前 Orderly 支持的所有功能，使用时只需要简单配置即可。

## Installation

```
npm install -S @orderly-network/core @orderly-network/react
```

- `@orderly-network/core` 是 Orderly 的核心库，包含了 Orderly 的核心逻辑，以及一些常用的工具函数。
- `@orderly-network/react` 是 Orderly 的 React 组件库，包含了一些常用的 React 组件，以及一些常用的 React Hooks。

## Usage

```tsx
<OnboardConnectorProvider>
  <OrderlyProvider {...config}>
    <TradingPage />
  </OrderlyProvider>
</OnboardConnectorProvider>
```
